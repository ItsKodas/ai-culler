[h1]v3.8.0[/h1]
[list]
[*] Switched surface intersection from FIRE LOD to VIEW LOD in both aic_client and aic_main. View Geometry is correct for visual occlusion; Fire LOD is ballistic penetration geometry and produced wrong results for many objects
[*] Fixed terrain-baked objects (trees, rocks) incorrectly occluding units. [b]lineIntersectsSurfaces[/b] returns a null object for things baked into terrain, null refs are now treated as non-blocking
[*] Fixed remote-controlled units (Zeus RC) being hidden by aic_client. Any unit currently being remote controlled is always shown regardless of LOS result
[*] Fixed all units near the player being hidden while remote controlling a unit via Zeus. The client renderer now detects when RC is active and suspends occlusion entirely for that tick
[*] Fixed remote-controlled units being culled by aic_main. Units with an active remote controller are excluded from the managed pool
[*] Fixed parachuting and free-falling units being culled by aic_main. Units with a downward velocity are excluded, covering both units inside a parachute and the free-fall window between exiting an aircraft and chute deployment
[*] Fixed backspace closing Zeus while typing in configuration edit boxes. The Zeus display now consumes the backspace key only when an edit field has keyboard focus
[*] Fixed "Generic error in expression" in the main loop LOS check
[*] Added terrain pre-check to aic_main's per-player LOS loop, consistent with aic_client's two-tier approach. Skips the expensive surface check when terrain already blocks the line
[*] Added color-coded Active count in the Zeus status window. Green below cap, yellow at cap, orange up to 2x cap, flashing red above
[*] Added [b]Protected from Culler[/b] checkbox to Eden Editor unit attributes under Object: States. No scripting needed
[*] Fixed conflict with the Hide Zeus module. aic_client was calling hideObject false on any visible unit, overriding the global hide and making the Zeus player visible to others. The renderer now only reveals units it hid itself
[*] Fixed enemy vehicle crews not triggering combat activation. Units inside vehicles were invisible to the infantry combat scan; a separate vehicle scan now covers armour, aircraft, and boats
[*] Fixed AI culled before boarding a vehicle staying simulation-disabled while seated. A dedicated pass re-enables any culled unit inside a vehicle on every tick
[*] Declared CBA as a hard dependency for both addons. The A3 launcher will now enforce CBA is loaded before the mod runs
[*] Fixed aic_client MP loop hiding AI seated inside vehicles. The SP loop already excluded vehicle crew but the MP loop was missing the same filter
[/list]

[h1]v3.7.0[/h1]
[list]
[*] Rewrote aic_client with separate singleplayer and dedicated server MP paths. Listen server (host-client) is explicitly unsupported
[*] Two-tier LOS: terrain check runs first as a cheap early-out, full surface intersection only runs for units within 600m that passed the terrain check
[*] Added [b]AIC_clientSurfaceRadius[/b] (default 600m), configurable via CBA Addon Options
[*] Camo nets and sandbag positions no longer occlude units. Class name filter updated to exclude objects with "net", "bag", or "bunker" in their type
[*] Switched surface intersection to eye-level positions for accurate origin points
[*] ADS cone reveal: units within ~30° of aim direction are force-revealed while aiming down sights or through an optic
[*] Safe radius default raised from 75m to 150m
[*] Zeus camera bypass now uses the curator display check, units are unhidden as the sweep processes them while Zeus is open
[*] Enable/disable notification popups now off by default, opt-in via Addon Options
[*] Added [b]AIC_clientCorpseRadius[/b] (default 300m). Dead AI beyond this distance are hidden by distance alone, no LOS check. Set to 0 to always show corpses
[/list]

[*] Bidirectional combat detection: when unit A detects enemy B in combat radius, both groups are forced active in the same tick

[*] [b]allUnits[/b] now snapshotted once per tick and reused throughout, eliminating redundant calls
[*] Added [b]Show Notifications[/b] to CBA Addon Options
